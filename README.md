# Disney-plus App

| React 사용한 Disney-plus OTT 홈페이지 입니다.

### 프로젝트 실행 방법

```
npm i
npm start
```

### Description

/api/axio.js 파일에는 API 호출에 필요한 키값이 설정되어 있어 업로드에 제외하였습니다.
